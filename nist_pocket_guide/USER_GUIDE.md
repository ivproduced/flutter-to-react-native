# NIST Pocket Guide - User Instructions

This guide explains how to use the main features of the NIST Pocket Guide app.

---

## 1. Getting Started

- **Install Flutter** (if running from source):
  - Download and install the [Flutter SDK](https://flutter.dev/docs/get-started/install).
  - Run `flutter pub get` in the project directory.
  - Start the app with `flutter run`.
- **Download from Store** (if available):
  - Install the app from your platform's app store (Android/iOS/Web/Windows/macOS/Linux).

---

## 2. Main Features & How to Use Them

### A. NIST 800-53 Controls
- **Browse Controls:**
  - Open the app and select "Controls" from the main menu.
  - Browse by control family or search for a specific control.
  - Tap a control to view details, guidance, and enhancements.
- **Favorites & Notes:**
  - Tap the star icon to add a control to your favorites.
  - Add personal notes using the notes section on each control detail page.
- **Recent Controls:**
  - Access recently viewed controls from the home screen or controls menu.

### B. AI RMF Playbook
- **Browse by Type or Topic:**
  - Select "AI RMF" from the main menu.
  - Filter entries by type (Govern, Map, Manage, Measure) or by topic.
  - Tap an entry for detailed information, references, and actors.

### C. NISTBot Chat (Beta)
- **Start a Chat:**
  - Select "NISTBot Chat" from the main menu.
  - Type your question about NIST controls, AI RMF, or compliance.
  - View AI responses and follow up as needed.
  - Note: Daily message limits may apply.

### D. System & SSP Management (Alpha)
- **Create a System:**
  - Go to "Systems" or "SSP" in the main menu.
  - Tap "+" to add a new information system.
- **Generate/Edit SSP:**
  - Select a system and choose "Generate SSP" or "Edit SSP".
  - Fill in control implementations, add evidence, and track status.

### E. Personalization & Settings
- **Theme:**
  - Change between light, dark, or system theme in the settings menu.
- **Preferences:**
  - Adjust user preferences and app behavior in settings.

## 3. Tips & Shortcuts
- Use the search bar to quickly find controls or AI RMF entries.
- Mark important controls as favorites for quick access.
- Use notes to document organization-specific guidance or implementation details.
- Check the about screen for app version and contact info.

---

## 4. Troubleshooting
- If the app fails to load data, check your internet connection (for AI chat).
- For persistent issues, restart the app or reinstall.
- For support, contact the project maintainer (see README).
